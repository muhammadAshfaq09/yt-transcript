One-Time Setup Process

This page allows you to convert an existing user account to an admin account. You'll need:

  • The email and password of an existing user account
  • The admin secret code (default: youtube-transcript-admin-secret)

After setup, you will be logged in and redirected to the admin dashboard.

Important: This setup page should be disabled after the first admin is created!
