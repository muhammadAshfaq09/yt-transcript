YouTube Transcript Converter API

API Version: 1.0.0

Welcome to the YouTube Transcript Converter API documentation. Our API allows you to programmatically convert YouTube videos to text transcripts, helping you integrate this functionality into your own applications.

Base URL

https://api.yttranscript.com/v1

For testing purposes, you can also use our development endpoint:

https://dev.yttranscript.com/v1

Getting Started

To use our API, you'll need to:

  1. Sign up for an account and subscribe to a plan that includes API access (Pro or Business plan)
  2. Generate an API key from your account dashboard
  3. Include your API key in the request headers as shown in the Authentication section

Authentication

All API requests require authentication using an API key. Include your API key in the X-API-Key header with each request.

curl --request GET \
  --url https://api.yttranscript.com/v1/transcript/VIDEO_ID \
  --header 'X-API-Key: your_api_key'
const options = {
  method: 'GET',
  headers: {
    'X-API-Key': 'your_api_key'
  }
};

fetch('https://api.yttranscript.com/v1/transcript/VIDEO_ID', options)
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));
import requests

headers = {
    'X-API-Key': 'your_api_key'
}

response = requests.get('https://api.yttranscript.com/v1/transcript/VIDEO_ID', headers=headers)
data = response.json()
print(data)

Important

Keep your API key secure and do not share it publicly. If your API key is compromised, you can generate a new one from your account dashboard and invalidate the old one.

Rate Limits

To ensure service stability, we enforce rate limits on API requests:

Plan Rate Limit Daily Limit
Pro 60 requests per minute 1,000 requests per day
Business 300 requests per minute 10,000 requests per day
Enterprise Custom Custom

Rate limit information is included in the response headers:

  • X-RateLimit-Limit: Maximum requests per minute
  • X-RateLimit-Remaining: Remaining requests in the current window
  • X-RateLimit-Reset: Time in seconds until the rate limit resets

Note

If you need higher rate limits, please contact our sales team for Enterprise pricing options.

Endpoints

Get Transcript

GET /transcript/{videoId}

Retrieves the transcript for a YouTube video.

Path Parameters

Parameter Type Description
videoId string The YouTube video ID (e.g., dQw4w9WgXcQ)

Query Parameters

Parameter Type Default Description
language string null Optional. Language code (e.g., en, es, fr). If not provided, uses the video's default language.
format string text Optional. Output format: text, json, srt, or vtt

Response

{
  "success": true,
  "transcript": "[00:00] First line of transcript\n[00:05] Second line of transcript...",
  "metadata": {
    "videoId": "dQw4w9WgXcQ",
    "title": "Video Title",
    "language": "en",
    "duration": 213
  }
}

Example Request

GET https://api.yttranscript.com/v1/transcript/dQw4w9WgXcQ?format=json

Get Video Information

GET /video-info/{videoId}

Retrieves metadata about a YouTube video.

Path Parameters

Parameter Type Description
videoId string The YouTube video ID

Response

{
  "success": true,
  "title": "Video Title",
  "author": "Channel Name",
  "thumbnail": "https://i.ytimg.com/vi/dQw4w9WgXcQ/hqdefault.jpg"
}

User History Endpoints

These endpoints allow you to manage a user's transcript history.

GET /history

Retrieves the user's transcript history.

POST /history/add

Adds a video to the user's transcript history.

Request Body

{
  "videoId": "dQw4w9WgXcQ",
  "videoTitle": "Video Title"
}

Error Handling

The API uses standard HTTP status codes to indicate the success or failure of a request. In case of an error, the response body will include a descriptive message.

Status Code Description
200 OK The request was successful
400 Bad Request The request was invalid or malformed
401 Unauthorized API key is missing or invalid
403 Forbidden The API key doesn't have permission for the requested operation
404 Not Found The requested resource was not found
429 Too Many Requests Rate limit exceeded
500 Internal Server Error An error occurred on the server

Error Response Example

{
  "success": false,
  "message": "No transcript found for this video",
  "error": "TRANSCRIPT_NOT_FOUND"
}

Code Examples

Node.js Example

const axios = require('axios');

async function getTranscript(videoId) {
  try {
    const response = await axios.get(`https://api.yttranscript.com/v1/transcript/${videoId}`, {
      headers: {
        'X-API-Key': 'your_api_key'
      }
    });
    
    return response.data;
  } catch (error) {
    console.error('Error fetching transcript:', error.response?.data || error.message);
    throw error;
  }
}

// Usage
getTranscript('dQw4w9WgXcQ')
  .then(data => console.log(data))
  .catch(error => console.error('Failed to get transcript'));

Python Example

import requests

def get_transcript(video_id, api_key):
    headers = {
        'X-API-Key': api_key
    }
    
    response = requests.get(f'https://api.yttranscript.com/v1/transcript/{video_id}', 
                           headers=headers)
    
    if response.status_code == 200:
        return response.json()
    else:
        print(f"Error: {response.status_code}")
        print(response.text)
        return None

# Usage
api_key = 'your_api_key'
result = get_transcript('dQw4w9WgXcQ', api_key)
print(result)
